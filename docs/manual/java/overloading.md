## Überladen von Methoden

Überladen bedeutet, dass derselbe Methodenname mehrfach in einer Klasse
verwendet werden kann. Damit das Überladen möglich ist, muss wenigstens eine der
folgenden Vorraussetzungen erfüllt sein:

1. Der Datentyp mindestens eines Übergabeparameters ist anders als in den übrigen
   gleichnamen Methoden.
2. Die Anzahl der Übergabeparameter ist unterschiedlich.[^java-tutorial]

[^java-tutorial]: https://www.java-tutorial.org/ueberladen_von_methoden.html
